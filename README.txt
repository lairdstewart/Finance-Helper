# finance_helper

Project to create a program to take the xlsx sheet from my bank statement and return summaries, information, and graphs.

First project with python oop, python regex, reading xlsx files in python

FinanceSummary is a class that takes a bank report xlsx's file name and provides: summary, net balance, net deposits/withdraws, avg withdraw/deposit all monthly or total. Create a scatter plot vs. time of expenses, and a histogram of withdraws.
